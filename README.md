## 🏥 Hospital Appointment Booking System

# 📌 Project Overview

The **Hospital Appointment Booking System** is a web-based application designed to simplify the process of booking medical appointments. It provides a modern and user-friendly interface where patients can easily schedule checkups and consultations with doctors.

---

# ✨ Features

* 🧑‍⚕️ Book appointments easily
* 📅 Select date and time slots
* 🏥 Choose type of service (Checkup / Consultation)
* ✅ Smart confirmation messages
* ⚠️ Error handling for invalid inputs
* 📊 Appointment summary display
* 🎨 Modern UI with icons and animations
* ⏳ Loading effects for better user experience

---

# 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
